# PacketScope

**PacketScope** — это учебное desktop-приложение для захвата, анализа, фильтрации и сохранения сетевого трафика.  
Проект разработан на языке **Python** с использованием принципов **объектно-ориентированного программирования**.

Приложение является упрощённым аналогом сетевого анализатора трафика и может использоваться для изучения сетевых пакетов, протоколов TCP/IP и базовой диагностики сетевой активности.

---

## Возможности приложения

PacketScope позволяет:

- захватывать сетевые пакеты в реальном времени;
- выбирать сетевой интерфейс для анализа;
- использовать BPF-фильтры до начала захвата;
- отображать пакеты в таблице;
- подсвечивать разные типы трафика цветами;
- определять популярные протоколы по портам:
  - HTTP;
  - HTTPS;
  - DNS;
  - SSH;
  - FTP;
  - SMTP;
  - POP3;
  - IMAP;
  - RDP;
  - DHCP;
  - NTP;
  - MySQL;
  - PostgreSQL;
  - Redis;
- просматривать подробную структуру выбранного пакета;
- просматривать payload в режимах:
  - ASCII;
  - HEX;
  - Raw Scapy;
- сохранять захваченный трафик в форматы:
  - `.pcap`;
  - `.pcapng`;
- открывать ранее сохранённые `.pcap` и `.pcapng` файлы;
- фильтровать уже захваченные пакеты по протоколу, IP-адресу, порту или тексту;
- отображать статистику трафика;
- показывать топ IP-адресов;
- показывать DNS-домены;
- выполнять простой анализ подозрительной активности;
- экспортировать отчёты в:
  - `.csv`;
  - `.txt`.

## Назначение проекта

Проект создан в рамках курсовой работы по дисциплине **«Объектно-ориентированное программирование»**.

В приложении используются основные принципы ООП:

- **инкапсуляция** — логика захвата, анализа, фильтрации, сохранения и интерфейса разделена по классам;
- **абстракция** — пользователь работает с графическим интерфейсом, не взаимодействуя напрямую с низкоуровневыми сетевыми механизмами;
- **композиция** — главное окно приложения объединяет объекты анализатора, сниффера, фильтра, статистики и хранилища;
- **модульность** — каждый файл проекта отвечает за отдельную часть функциональности.
